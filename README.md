Interva Sambi Kopi POS Files

Package name:
- interva.sambikopi

Available pages:
- Orders: muted cream/peach interface, search, filter, refresh, complete, assign, and cancel order actions.
- Menu: add, edit, delete, search, refresh menu data, and IDR price formatting.
- Inventory: search, check stock, add stock, notify owner, and stock status display.

Under development:
- Dashboard

How to use:
1. Run with: mvn javafx:run
3. Open menu.fxml, inventory.fxml, or orders.fxml in Scene Builder if you want to edit the UI.

Notes:
- Data is in-memory sample data, not database data.
- Inline FXML styles are used, no external CSS.
- All main buttons have fx:id and onAction.
- Search fields are functional at runtime.
